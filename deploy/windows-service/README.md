# Windows Service Deployment Assets

This folder contains the artifacts used for running the API as a Windows service:

- `bible_server.exe` (built executable)
- `bible_server_service.exe` (WinSW wrapper)
- `bible_server_service.xml` (WinSW config)
- `start_bible_server.bat` / `start_bible_server_highperf.bat`
- `bible_server_task.xml` / `bible_server_highperf_task.xml` (Task Scheduler samples)
- `WINDOWS_SERVICE.md` (setup instructions)

If you want to rebuild the executable, see `bible_server.spec`.
